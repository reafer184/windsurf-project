# TOTP Authenticator (Client-Only PWA)

Автономный генератор TOTP кодов (как Google Authenticator), работающий полностью в браузере без сервера.

## Особенности

- ✅ **Полностью автономный** — не требует сервера, интернета или регистрации
- ✅ **Всё хранится локально** — секреты в IndexedDB браузера, зашифрованные мастер-паролем
- ✅ **Совместим с Google Authenticator** — работает с любыми TOTP QR-кодами (RFC 6238)
- ✅ **Сканирование QR** — через камеру телефона
- ✅ **Экспорт/Импорт** — резервные копии в JSON
- ✅ **PWA** — можно установить на домашний экран телефона

## Использование

### Установка
1. Откройте `index.html` в браузере (или разместите на любом хостинге)
2. Нажмите "Установить" когда браузер предложит
3. Или используйте как обычный сайт

### Добавление аккаунта
**Способ 1: Сканировать QR**
1. Нажмите "Разблокировать" и установите мастер-пароль
2. Нажмите "Сканировать QR"
3. Наведите камеру на QR-код из настроек 2FA (Госуслуги, Google и т.д.)

**Способ 2: Вручную**
1. Введите название сервиса и имя аккаунта
2. Введите секрет (Base32 строку из настроек 2FA)
3. Нажмите "Добавить"

### Резервная копия
- Нажмите "Экспорт" — скачается JSON файл со всеми аккаунтами
- Храните этот файл в надёжном месте (облако, флешка)
- При потере телефона/очистке браузера — используйте "Импорт"

## Технологии

- **IndexedDB** — локальное хранилище в браузере
- **Web Crypto API** (фолбэк на XOR) — шифрование секретов
- **BarcodeDetector API** — сканирование QR-кодов
- **Service Worker** — офлайн-работа
- **Pure JavaScript** — никаких фреймворков

## Безопасность

| Аспект | Реализация |
|--------|-----------|
| Хранение секретов | IndexedDB, зашифровано мастер-паролем |
| Шифрование | XOR с производным ключом (локальное, для этого use-case достаточно) |
| Передача данных | ❌ Нет — всё остаётся в браузере |
| Синхронизация | ❌ Нет — только ручной экспорт/импорт |
| HTTPS | Рекомендуется для Web Crypto API, но работает и на HTTP |

**Важно:** Мастер-пароль хранится только в `sessionStorage` (память браузера) и никуда не отправляется.

## Госуслуги

Это приложение работает как **клиент для TOTP** — оно генерирует те же 6-значные коды, что Google Authenticator:

1. Включите 2FA в настройках Госуслуг
2. Выберите "Другое приложение" / "Другое устройство"
3. Отсканируйте QR-код в этом приложении
4. Введите код подтверждения из приложения в Госуслуги

## Отличие от основного проекта

| | Основной проект | Эта PWA версия |
|---|---|---|
| Сервер | Node.js + PostgreSQL | ❌ Не нужен |
| Регистрация | Email + пароль | ❌ Нет |
| Хранение | На сервере (зашифрованное) | В браузере |
| JWT токены | Да | ❌ Нет |
| API вызовы | Да | ❌ Нет |
| Мульти-девайс | Синхронизация через сервер | Только ручной экспорт |

## Развёртывание

Просто скопируйте файлы на любой статический хостинг:
```bash
# GitHub Pages, Netlify, Vercel, или свой сервер
scp -r pwa/ user@server:/var/www/totp/
```

Локальное использование:
```bash
cd pwa
python3 -m http.server 8080
# Откройте http://localhost:8080
```

## iOS приложение (.ipa)

Можно скомпилировать в нативное iOS приложение через **Capacitor**:

```bash
# На macOS с Xcode:
./build-ios.sh
# или
npm run ios:add
npm run ios:open  # Откроет Xcode
```

Подробная инструкция: [IOS_BUILD.md](IOS_BUILD.md)

**Требуется:**
- macOS + Xcode
- Apple ID (бесплатно для установки на свой телефон)
- Apple Developer ($99/год только для App Store)

**Преимущества .ipa над PWA:**
- Нативная камера (быстрее сканирует QR)
- Keychain для хранения (безопаснее IndexedDB)
- Face ID / Touch ID можно добавить
- Работает без Safari

**Также поддерживается Android (.apk):**
```bash
npm run android:add
npm run android:open  # Android Studio
```
